Bills

The Bills Option allows the user to add two user defined paydays and an unlimited amount of bills. When the user is ready, they can hit the done button to run the program and see its output on the next page.

Debts

The Debts Option allows the user to declare their income after bills, add an unlimited amount of user defined debts, and then finally run the program and view its output.

Both

As you may have already surmised, the both option allows the user to run both the bills and debt options one after the other. The 'leftover' amount after bills will automagically be used as 'income' for a 'debts' run.
